Used Car Price Prediction Model

I developed a predictive model using data science and machine learning to estimate used car prices. This involved gathering, preprocessing, and analyzing a dataset with key attributes like car make, model, year, mileage, and location. I engineered features such as car age and mileage per year. By applying algorithms like linear regression, decision trees, and random forests, I built predictive models, optimizing their performance through techniques like cross-validation and hyperparameter tuning. I evaluated model accuracy using metrics like mean absolute error and root mean squared error, and communicated insights effectively through visualizations.

Mushroom classifier

I developed a mushroom classifier using the VGG convolutional neural network (CNN) architecture to categorize images of mushrooms into different species. This involved curating and preprocessing a dataset of mushroom images, including edible and poisonous varieties, and augmenting it for increased diversity. Implementing the VGG model, I leveraged transfer learning by fine-tuning the pre-trained model on the mushroom dataset to improve classification accuracy. I evaluated the classifier's performance using metrics such as accuracy, precision, recall, and F1-score to ensure its effectiveness. Finally, I deployed the classifier as a user-friendly web application, allowing users to identify mushroom species from images with ease.

Book Store Chatbot with Natural Language Processing (NLP)

I developed a chatbot using natural language processing (NLP) models to assist customers in a book store. This involved training machine learning models to understand and respond to customer queries regarding book titles, authors, genres, and availability. By integrating the chatbot with the store's database, it provided real-time information on book inventory and pricing. The chatbot utilized techniques such as intent recognition, entity extraction, and dialogue management to improve its conversational capabilities. Through user testing and iterative improvements based on feedback, I ensured the chatbot provided accurate and helpful responses, ultimately enhancing the customer experience.
